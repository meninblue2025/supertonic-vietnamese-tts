# Supertonic TTS tiếng Việt — script chạy local

Script Python để chạy Supertonic TTS tiếng Việt offline trên Mac M1.
10 giọng (5 nam, 5 nữ). Miễn phí, không cần internet sau khi cài đặt.

## Cài đặt

1. Cài Supertonic từ: [link Supertonic GitHub gốc]
2. Tải file `noi.py` ở repo này
3. Chạy: `python noi.py`
